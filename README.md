# KeenKeeper — Keep Your Friendships Alive

## 📌 Project Name
KeenKeeper

## 📝 Description
A simple web app to track your personal relationships.  
You can manage friends, track interactions, and stay connected easily.

## ⚙️ Technologies Used
- React
- React Router
- Tailwind CSS + DaisyUI
- React Toastify
- Recharts

## ⭐ Key Features
- Quick Check-In (Call, Text, Video)
- Timeline with filter option
- Analytics with chart visualization
